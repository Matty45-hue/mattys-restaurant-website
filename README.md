# 🍽️ Matty’s Restaurant Website

A modern and responsive **restaurant website** designed for showcasing food, ambiance, and services of a restaurant. Built using **HTML, CSS, and JavaScript**, this project delivers a clean and engaging user experience.

---

## 🚀 Features

* 🍔 Eye-catching homepage with hero section
* 📜 Menu section displaying food items
* 🧑‍🍳 About section for restaurant details
* 📞 Contact section with essential information
* 📱 Fully responsive design (mobile-friendly)
* 🎨 Smooth animations and modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run the Project

### ✅ Method 1: Direct Open

1. Download or clone the repository
2. Extract the ZIP file (if downloaded)
3. Open `index.html` in any browser

---

### ✅ Method 2: Using VS Code (Recommended)

1. Open the project folder in VS Code
2. Install **Live Server Extension**
3. Right-click `index.html`
4. Click **"Open with Live Server"**
